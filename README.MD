쿠팡 상품 URL을 입력하면 상세페이지를 분석해 점수를 매겨주는 Streamlit 앱을 만들 거야. Google Gemini 1.5 Flash API를 메인 엔진으로 사용해줘.

기술적 요구사항:

이미지 추출: requests와 BeautifulSoup을 사용하여 입력된 쿠팡 URL에서 메인 썸네일 이미지와 상세페이지 이미지(desc-img) URL을 크롤링한다.

Gemini 연동: google-generativeai 라이브러리를 사용한다. 추출된 이미지들을 Gemini 모델에 전달하여 분석한다.

분석 프롬프트(LLM 전송용): Gemini에게 다음과 같은 페르소나를 부여해줘. "너는 10년 차 이커머스 상세페이지 컨설팅 전문가야. 다음 이미지를 보고 '썸네일 클릭률', '상단 후킹 지수', '구매 전환 설득력', '가독성' 4개 항목을 각각 25점 만점으로 평가해줘. 총합 100점 만점으로 계산하고, 구체적인 개선안을 불렛포인트로 작성해."

결과 시각화: >    - st.columns를 활용해 좌측엔 분석한 이미지를, 우측엔 Gemini의 분석 결과를 배치해줘.

총점은 큰 글씨로 강조하고, 항목별 점수는 막대그래프(st.bar_chart)로 표현해줘.

보안: API Key는 st.secrets에서 불러오도록 작성해줘."
